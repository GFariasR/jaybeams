16 /**
17  * A helper type to define short (and fixed sized) string fields.
18  *
19  * The ITCH-5.0 protocol uses many fields that are short strings, that
20  * is, fixed-length alpha numeric fields. While it is normally easy
21  * to treat these fields as character strings, the fields may not be
22  * NUL terminated. In fact, the protocol spec specifically states
23  * that the fields will be left justified, and padded with spaces.
24  *
25  * We want an in-memory representation that is NUL terminated (for
26  * compatibility with all the C/C++ libraries), and we do not wish to
27  * allocate memory for these fields (as we would do if we used
28  * std::string).
29  *
30  * @tparam wire_size_value the size of the field on the wire, in
31  * bytes.
32  * @tparam value_validator a functor to validate the value for those
33  * fields that are only supposed to assume a known set of values.
34  */
