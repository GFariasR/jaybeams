Detailed Description

template<typename container_t, typename value_t>
class jb::fftw::tde_result< container_t, value_t >

tde_result is a type to handle the result of operating over the last dimension array of a multi array of timeseries.

This is used when the operation returns a value out of the array, therefore tde_result has one dimension less than the tde_result constructor's argument. Handles TDE result for container type timeseries.

Template Parameters
container_tcontainer type timeseries
value_ttype to store the result of the timeseries operation
