10 /**
11  * Keep a histogram of observed event rates.
12  *
13  * This class composes a histogram and an event_rate_estimator to
14  * collect running statistics about event rates. While an
15  * event_rate_estimator computes the number of events in the last X
16  * seconds, this class keeps a tally of the observed event rates,
17  * which can be used to build median, maximum and any other quantile
18  * of the message rate.
19  *
20  * The user specifies the maximum message rate to keep at full
21  * resolution, any rate above that value is simply recorded in the
22  * histogram overflow bucket.
23  *
24  * @tparam counter_type The type used in the histogram counters. For
25  * most applications @a int is adequate, but you may use a smaller
26  * type if (a) you are memory constrained, and (b) you do not expect
27  * high values in the counters.
28  * @tparam duration_type Define the bucket size for the
29  * event_rate_estimator.
30  */
