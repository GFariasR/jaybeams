Detailed Description

template<typename counter_type = int, typename duration_type = std::chrono::microseconds>
class jb::event_rate_histogram< counter_type, duration_type >

Keep a histogram of observed event rates.

This class composes a histogram and an event_rate_estimator to collect running statistics about event rates. While an event_rate_estimator computes the number of events in the last X seconds, this class keeps a tally of the observed event rates, which can be used to build median, maximum and any other quantile of the message rate.

The user specifies the maximum message rate to keep at full resolution, any rate above that value is simply recorded in the histogram overflow bucket.

Template Parameters:
counter_typeThe type used in the histogram counters. For most applications int is adequate, but you may use a smaller type if (a) you are memory constrained, and (b) you do not expect high values in the counters.
duration_typeDefine the bucket size for the event_rate_estimator.
